# {{serverName}}

An MCP server generated by **create-mcp-server** from an OpenAPI spec.
Every endpoint in the API became one MCP tool that calls the real API.

- **Base URL:** `{{baseUrl}}`

## Run it

1. Install dependencies:
   `npm install`
2. Start in dev mode:
   `npm run dev`

## Configuration (environment variables)

- `API_BASE_URL` — override the API base URL (defaults to the one above).
- `API_TOKEN` — if set, it is sent on every request as `Authorization: Bearer <token>`.

Example (Windows PowerShell):

```powershell
$env:API_TOKEN = "your-key-here"
npm run dev
```

## Test it in the MCP Inspector

`npm run build`
`npx @modelcontextprotocol/inspector node dist/index.js`

Open the URL it prints, pick a tool (one per endpoint), fill in the inputs,
and run it — the response from the live API comes back as text.

## Notes

- Tool inputs are passed as strings. For endpoints that take a JSON body,
  pass the `body` argument as a JSON string.
