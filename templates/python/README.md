# {{serverName}}

An MCP server generated by **create-mcp-server**.
It exposes one example tool: `{{toolName}}`.

## Run it (pip + virtual environment)

1. Create and activate a virtual environment:
   - **Windows:** `python -m venv .venv` then `.venv\Scripts\activate`
   - **macOS/Linux:** `python -m venv .venv` then `source .venv/bin/activate`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Start the server:
   `python server.py`

## Run it (uv — faster, optional)

`uv run --with "mcp[cli]" server.py`

## Test it in the MCP Inspector

The Inspector is a visual tool to try your server:

`npx @modelcontextprotocol/inspector python server.py`

Open the URL it prints, find the `{{toolName}}` tool, type a name, and run it.
