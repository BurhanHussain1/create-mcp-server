# {{serverName}}

An MCP server generated by **create-mcp-server**.
It exposes one example tool: `{{toolName}}`.

## Run it

1. Install dependencies:
   `npm install`
2. Start in dev mode (runs the TypeScript directly):
   `npm run dev`

## Build for production

`npm run build` then `npm start`

## Test it in the MCP Inspector

The Inspector is a visual tool to try your server. Build first, then point
the Inspector at the compiled file (this keeps the stdio channel clean):

`npm run build`
`npx @modelcontextprotocol/inspector node dist/index.js`

Open the URL it prints, find the `{{toolName}}` tool, type a name, and run it.
